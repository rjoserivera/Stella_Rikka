📖 Instrucciones de Sintaxis (3 formas soportadas)

  1. SQL (CREATE TABLE):
    CREATE TABLE Usuarios ( id INT, nombre VARCHAR );
  2. DBML:
    Table Pedidos { id int, usuario_id int } Ref: Pedidos.usuario_id > Usuarios.id
  3. Lista Simple (Markdown-like):
    Usuarios
    - id
    - nombre
    Pedidos
    - usuario_id -> Usuarios

✨ Funciones Principales

  • Importar BD: Genera tablas y relaciones automáticamente insertando SQL o sintaxis DBML.
