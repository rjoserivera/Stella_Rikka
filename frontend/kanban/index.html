❓ ¿Cómo usar el Tablero Kanban?

El Tablero Kanban es una herramienta visual para gestionar el flujo de trabajo. Te permite ver exactamente en qué estado se encuentra cada tarea de tu proyecto, identificar cuellos de botella y maximizar la eficiencia del equipo.

🔴 Por Hacer (Backlog)

Lista de todas las tareas pendientes. Es el "almacén" de cosas que aún no han comenzado. Prioriza las tarjetas colocando las más importantes arriba.

🟠 En Progreso

Tareas que el equipo está desarrollando en este momento. Intenta no saturar esta columna; el foco es clave para terminar rápido.

🔵 En Pruebas

El desarrollo terminó, pero se está verificando que cumpla con la calidad esperada (QA) antes de darlo por finalizado.

🟢 Terminado

Tareas finalizadas con éxito y listas para producción. ¡Es la métrica principal del avance de tu proyecto!

🖱️ Cómo usar esta aplicación

  • Usa el botón "+" debajo de una columna o el botón "+" inferior derecho para crear tareas nuevas.
  • Arrastra y suelta (Drag & Drop) las tarjetas entre las columnas para avanzar su estado de desarrollo.
  • Haz clic sobre cualquier tarjeta para editar su contenido, cambiar el tipo (Feature, Bug) o la prioridad.
  • Abre el "📊 Resumen" en la barra superior para ver de forma gráfica cómo se distribuye la carga de trabajo.
