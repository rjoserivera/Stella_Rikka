❓ ¿Cómo usar el Editor de Diagramas?

El Editor de Diagramas es una herramienta para crear diagramas de flujo y arquitectura. Puedes visualizar procesos, componentes o bases de datos de forma rápida y sencilla.

▬ Proceso (Rectángulo)

Representa una acción, cálculo o paso genérico dentro de tu sistema. Es la figura más utilizada.

◆ Decisión (Rombo)

Representa una condición o pregunta lógica (Ej: ¿Usuario está logueado?). Generalmente tiene dos salidas (Sí / No).

⬭ Inicio/Fin (Óvalo)

Marca el punto de partida o de finalización de un proceso o subproceso.

🗄️ Base de Datos (Cilindro)

Indica almacenamiento de datos persistentes. Ideal para diagramas de arquitectura.

🖱️ Controles Básicos

  • Añadir Nodos: Arrastra (Drag & Drop) las figuras desde el panel izquierdo hacia el lienzo.
  • Conectar: Selecciona el modo "Conectar" a la izquierda. Haz clic en el nodo de inicio y luego en el nodo destino.
  • Editar: Haz clic sobre cualquier nodo o línea para abrir sus propiedades a la derecha (Color, tamaño, texto).
  • Mover Canvas: Mantén presionada la barra espaciadora (Space) o el botón central del ratón y arrastra.
  • Atajos: Suprimir / Backspace (borrar), Ctrl+C / Ctrl+V (copiar y pegar), Ctrl+Z (deshacer).
